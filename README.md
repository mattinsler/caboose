# Caboose

  A [coffeescript](http://coffeescript.org)-happy [express](http://expressjs.com/)-based server-side MVC framework based on [Rails](http://rubyonrails.org/)

## Installation

    npm install caboose

## Getting Started

    $ caboose new app_name
    $ cd app_name
    $ caboose server
